# To generate a default project 

```shell script
mvn archetype:generate \
-DarchetypeArtifactId=maven-archetype-quickstart  \
-DarchetypeVersion=1.4  \
-DinteractiveMode=false \
-DgroupId=org.qbeek   \
-DartifactId=PROJECT_NAME
```

- update Java version in pom.xml
